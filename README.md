# Nebulosity

## Overview

Nebulosity is an image capture and processing software designed for astrophotography on Windows systems. It provides tools for acquiring, calibrating, and stacking astronomical images, aiming to support users in producing clear and detailed celestial photos.

## Key Features

**Image capture from various astronomy cameras**  
**Calibration using dark, flat, and bias frames**  
**Image stacking and alignment capabilities**  
**Support for multi-channel imaging and filters**  
**Real-time preview and focus assistance**  
**Batch processing for multiple image sets**  
**Customizable settings for exposure and gain control**  

## Why Nebulosity?

Nebulosity offers a straightforward interface focused on reliability and clarity, making it suitable for astrophotographers who require consistent results. Its functionality emphasizes usability, providing practical tools that assist in image acquisition and processing without unnecessary complexity.

## Benefits

Nebulosity helps streamline the workflow of astrophotography by integrating capture and processing features in a single application. It supports various camera models and imaging techniques, facilitating improved image quality through calibration and stacking. The software design allows users to focus on image detail enhancement with practical controls and options.

## Compatibility

This repository is developed for Windows operating systems and delivers stable performance and efficient operation on compatible hardware. The software is optimized to work smoothly within the Windows environment, ensuring consistent results during image capture and processing tasks.
